# Feature Ideas

Living notes for future feature updates, organized by product area.

Each file captures raw ideas — not committed work. Add freely; prune as things ship or get dropped.

## Index

- [Account](account.md) — profile, sharing, settings
- [Planner](planner.md) — the weekly planner view
- [Camp list](camp-list.md) — browse/search (currently `/explore`)
- [Camp detail](camp-detail.md) — individual camp page
- [Kids](kids.md) — kid profiles
- [Onboarding](onboarding.md) — signup → first planner entry
- [Schedule](schedule.md) — shared/public schedule view
- [Submit](submit.md) — operator-facing submit flow
- [Cross-cutting](cross-cutting.md) — ideas that span multiple areas
- [General design/](general%20design/) — design-system ideas not tied to one product area (buttons, spacing, shadows, etc.)

## How to use

- Drop a bullet under the right file. No formatting rules.
- Include **why** when it matters — the motivation fades faster than the idea.
- Move an idea to a GitHub issue once it's ready to be worked on, then delete the bullet here.
