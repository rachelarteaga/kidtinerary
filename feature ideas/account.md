# Account

Profile, sharing, and account settings. Routes: `/account`, `/account/profile`, `/account/sharing`.

## Ideas

- **Link planners between users.** When kids across families share a camp, the shared kid/camp shows up on each linked user's planner automatically. Enables coordination (pickup, carpools, signing up together) without duplicate entry.
- **Broader "planning with friends" feature.** Beyond auto-linking on shared camps — let friends opt into seeing each other's planners (or specific weeks) so families can plan camps together.
